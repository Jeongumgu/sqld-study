## 1. 데이터 제어어(DCL)에 해당하는 명령어?
    - DML(Manipulation - 조작) : SELECT, INSERT, UPDATE, DELETE
    - DDL(Definition - 정의) : CREATE, ALTER, DROP, RENAME
    - DCL(Control - 제어) : GRANT, REVOKE
    - TCL(Transaction - 트랜잭션) : COMMIT, ROLLBACK
