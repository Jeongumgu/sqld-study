## 51. 서브쿼리의 종류
    - 단일 행 서브쿼리 : 메인 쿼리로 하나의 값이 반환됨
    - 다중 행 서브 쿼리 : 메인 쿼리로 여러 개의 값이 반환됨
    - 다중 컬럼 서브 쿼리 : 메인 쿼리로 여러개의 컬럼 값이 반환됨.
